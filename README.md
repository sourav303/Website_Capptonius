# Capptonius Technology INC Website

This is a static, deploy-ready website for Capptonius Technology INC.

## Front-end approach

- Static HTML for simple hosting
- Modern CSS for responsive layout, motion, glass panels, and SVG visuals
- Vanilla JavaScript for menu behavior, scroll reveal, counters, service-card interaction, and contact email flow
- No build step required
- No external CDN dependency required

## Files

- index.html - page content and structure
- styles.css - responsive visual design inspired by the provided Capptonius reference homepage
- script.js - mobile menu, animations, service interaction, and email inquiry behavior
- netlify.toml - Netlify static deploy settings
- vercel.json - Vercel static deploy settings

## Deploy

Upload the full contents of this folder to any static host, including GitHub Pages, Netlify, Vercel, Cloudflare Pages, or traditional web hosting.

The contact form opens the visitor's email app and sends to hr@capptonius.com, so no backend is required.


## UI enhancement notes

The homepage now includes a premium static UI layer: animated counters, scroll reveal effects, interactive service cards, a delivery-model tab section, and a moving technology capability strip. It still has no build step and no required paid tools.

## Sharing the website link

Upload this entire folder to a static host. The easiest options are Netlify Drop or Vercel. After upload, the host will give you a public link that you can share with clients.
